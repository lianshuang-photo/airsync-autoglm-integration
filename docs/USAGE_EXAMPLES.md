## 💡 使用技巧

### 1. 明确指令

❌ 不好的指令：
```
打开那个购物的
找一下东西
```

✅ 好的指令：
```
打开淘宝
在淘宝搜索iPhone 15
```

### 2. 分步执行

对于复杂任务，建议分步执行：

```
第一步：打开淘宝
第二步：搜索无线耳机
第三步：按销量排序
第四步：查看第一个商品详情
```

### 3. 使用具体名称

❌ 模糊：
```
给那个人发消息
```

✅ 具体：
```
给张三发微信消息
```

### 4. 避免敏感操作

AI 会自动识别敏感操作并请求确认：
- 支付操作
- 删除数据
- 修改重要设置
- 发送消息（会显示预览）

### 5. 善用快捷提示

在 AI Chat 界面中，点击示例提示可以快速输入常用指令。

## 🚫 限制说明

### 不支持的操作

1. **需要验证码的操作**
   - 登录账号
   - 支付确认
   - 短信验证

2. **需要人脸识别的操作**
   - 面部解锁
   - 支付验证

3. **复杂的图形操作**
   - 精确的绘图
   - 复杂的手势

4. **跨应用的复杂流程**
   - 从淘宝复制链接到微信分享（需要多次切换）

### 遇到问题时

如果 AI 无法完成任务：
1. 简化指令，分步执行
2. 检查应用是否已安装
3. 确认 ADB 连接正常
4. 查看错误提示信息

## 📊 性能建议

### 快速响应的任务

- 打开应用（1-2秒）
- 简单搜索（3-5秒）
- 点击按钮（1-2秒）

### 较慢的任务

- 复杂的多步骤操作（10-30秒）
- 需要等待页面加载（5-10秒）
- 需要滚动查找内容（10-20秒）

### 优化建议

1. 避免在网络慢时执行复杂任务
2. 确保手机性能良好
3. 关闭不必要的后台应用
4. 使用本地部署的模型（更快）
